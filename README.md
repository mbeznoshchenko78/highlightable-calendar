# Calendar Highlighter

A lightweight yearly calendar tool you can use directly in your browser.

## Features

- Choose any year
- Start weeks on Sunday or Monday
- Highlight dates in different colors
- Erase individual dates
- Clear the whole year
- Print the calendar
- Export the result as PDF
- Saves highlights locally in your browser

## Live Demo

Use the live app here:
[[Open Calendar Highlighter](YOUR_PAGES_LINK_HERE)](https://mbeznoshchenko78.github.io/highlightable-calendar/)

## How to Use

1. Select a year
2. Choose whether the week starts on Sunday or Monday
3. Pick a color and click or drag across dates
4. Use **Erase** to remove highlights
5. Use **Export PDF** to save the calendar

## Privacy

This tool runs entirely in your browser and stores highlights locally on your device.

## Tech

Single self-contained HTML file. No install required.
